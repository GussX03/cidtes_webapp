# CIDTES Energía y Sustentabilidad — Sitio Web Oficial

Sitio web oficial de **CIDTES Energía y Sustentabilidad**, organización interdisciplinaria con más de 20 años de experiencia en energía, sustentabilidad, seguridad y salud en el trabajo, y desarrollo organizacional.

## Descripción

CIDTES ofrece soluciones integrales a través de cinco líneas de servicio:

- **Capacitación** — Cursos y talleres especializados en energía, seguridad, desarrollo personal, formativo, valuación y desarrollo organizacional.
- **Certificación** — Procesos de certificación de competencias laborales avalados por el CONOCER.
- **Consultoría** — Asesoría especializada en gestión energética y desarrollo sustentable.
- **Investigación y Consultoría** — Estudios técnicos e investigación aplicada en energía renovable y sustentabilidad.
- **Cursos** — Catálogo completo de servicios y cursos disponibles por categoría.

## Tecnologías

- [Next.js 14](https://nextjs.org/) — Framework de React con App Router
- [Tailwind CSS](https://tailwindcss.com/) — Estilos utilitarios
- [TypeScript](https://www.typescriptlang.org/) — Tipado estático
- [Vercel](https://vercel.com/) — Despliegue en producción

## Instalación local

```bash
npm install
npm run dev
```

El servidor correrá en `http://localhost:3000`.
